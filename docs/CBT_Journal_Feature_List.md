# 📄 CBT 기반 마음 챙김 감정 기록 저널 (Thought Record) 기능 정의서

본 문서는 인지행동치료(CBT, Cognitive Behavioral Therapy) 이론을 기반으로 한 감정 기록 저널 서비스의 **1단계: 기능 정의서(Feature List)**입니다. 
이 프로젝트는 React(프론트엔드), Node.js(백엔드), PostgreSQL(데이터베이스)을 기술 스택으로 사용하며, 초보 개발자가 단계별로 차근차근 구현할 수 있도록 설계되었습니다.

---

## 1. 프로젝트 개요 (Project Overview)
* **프로젝트명**: 마음 챙김 감정 기록 저널 (Thought Record)
* **주요 타겟**: 스스로의 부정적 생각 패턴(인지 왜곡)을 인지하고, 합리적인 대안적 사고를 통해 감정을 조절하고자 하는 사용자
* **핵심 가치**: 
  1. 감정 변화의 정량적 측정 (사전/사후 감정 수치 기록)
  2. 왜곡된 생각(자동 사고)의 인지 및 이성적 반박(대안 사고) 훈련
  3. 직관적인 통계 차트를 통한 생각 패턴의 메타인지 제공

---

## 2. Must-Have (필수 구현 기능)
프로젝트 오픈 및 포트폴리오의 최소 기능 요구사항(MVP)으로, **반드시 구현해야 하는 핵심 기능**입니다.

### ① 회원 관리 (Authentication)

사용자가 안심하고 사적인 감정을 기록할 수 있도록 보호하는 기본 보안 장치입니다.

* **이메일 회원가입**: 
  * 입력 항목: 이메일(ID), 비밀번호, 비밀번호 확인, 닉네임
  * 비밀번호는 단방향 암호화(bcrypt) 처리하여 데이터베이스에 저장합니다.

* **이메일 로그인**:
  * 입력한 이메일과 비밀번호가 일치하는지 확인합니다.
  * 로그인 성공 시 **JWT(JSON Web Token)**를 발급하여 브라우저의 로컬 스토리지 또는 쿠키에 저장해 로그인 상태를 유지합니다.

* **로그아웃**:
  * 브라우저에 저장된 JWT 토큰을 만료/삭제 처리하여 안전하게 로그아웃합니다.

### ② 사전/사후 감정 강도 측정 & CBT 일기 작성 (CBT Diary)

이 서비스의 핵심인 인지행동치료 5단계 기록법을 디지털화한 기능입니다.

* **1단계: 사전 감정 및 강도 측정**
  * 현재 느끼는 감정 종류 선택 (예: 불안, 우울, 분노, 슬픔 등)
  * 감정의 강도를 슬라이더 또는 라디오 버튼으로 입력 (1점 ~ 10점)

* **2단계: 상황(Situation) 및 자동 사고(Automatic Thought) 기록**
  * **상황**: 어떤 일이 있었는지 구체적이고 객관적으로 기록 (텍스트 필드)
  * **자동 사고**: 상황이 발생했을 때 머릿속에 즉각적으로 스친 부정적인 생각 기록 (텍스트 필드)

* **3단계: 인지 왜곡(Cognitive Distortions) 유형 선택**
  * 자동 사고에 포함된 인지 왜곡 유형을 선택합니다. (중복 선택 가능)
  * *제공 유형 예시*: 흑백논리, 과도한 일반화, 정신적 필터, 감정적 추론, 파국화 등

* **4단계: 대안적 사고(Alternative Thought) 작성**
  * 원래 떠올랐던 부정적 생각(자동 사고)에 대해 스스로 질문을 던지고, 보다 이성적이고 합리적인 새로운 생각(반박)을 기록합니다. (텍스트 필드)

* **5단계: 사후 감정 및 강도 측정**
  * 합리적인 대안 생각을 작성한 후, 처음에 선택했던 동일한 감정의 강도가 어떻게 변했는지 다시 측정합니다. (1점 ~ 10점)

### ③ 일기 목록 및 상세 조회 (Diary List & View)

사용자가 이전에 작성한 글들을 다시 읽고 돌아보며 회복 탄력성을 얻는 공간입니다.

* **일기 목록 조회**:
  * 작성일 기준 **최신순** 정렬
  * 한 카드 안에 '작성일', '대표 감정 아이콘', '핵심 자동 사고', '사전 ➔ 사후 감정 변화(예: 불안 8점 ➔ 3점)' 정보를 직관적으로 요약 노출

* **일기 상세 보기**:
  * 목록에서 특정 일기 카드를 클릭했을 때 해당 일기의 모든 입력값(상황, 자동사고, 인지왜곡 유형, 대안사고, 감정 수치 변화)을 한 화면에 보여주는 상세 페이지 제공

### ④ 감정 통계 대시보드 (Dashboard)

누적된 일기 데이터를 가공하여 시각적 피드백을 주는 페이지입니다.

* **감정 변화 추이 그래프**:
  * 선 그래프(Line Chart)를 이용해 일기를 작성하기 전(사전)과 후(사후)의 감정 강도 평균치 변화를 시각화 (예: 평균 감정 감소율 45% 등으로 직관적 통계 제공)
  
* **주요 인지 왜곡 비율 차트**:
  * 도넛/원형 그래프(Pie Chart)를 이용해 사용자가 자주 선택했던 인지 왜곡 유형의 비율을 누적으로 보여줌 (예: 내 생각 패턴 중 '파국화'가 40%로 가장 높음 등)

기본 필수 기능 구현이 모두 성공적으로 끝난 뒤, 완성도를 높이기 위해 추가할 확장 기능들입니다.

| :--- | :--- | :--- | :--- |
| **로그인** | 소셜 로그인 | 구글, 카카오 등 간편 소셜 로그인 구현 | 중 (OAuth 2.0 연동) |

| **시각화** | 월별 감정 캘린더 | 달력 화면에 날짜별로 당일 작성한 대표 감정 아이콘을 도장처럼 표시 | 중 |
| **사용성** | 인지 왜곡 도움말 | 감정 일기 작성 화면에서 왜곡 유형 선택 시 쉬운 정의와 예시를 담은 툴팁 제공 | 하 (화면 레이아웃 작업) |
